Questions fréquemment posées

Si vous avez des questions, nous avons la réponse. Nous avons une liste de questions fréquemment posées à notre sujet.

YAWI est une solution 100% africaine créée par COK-SERVICE qui vous permet de transférer de l’argent d’un wallet vers n’importe quel autre wallet instantanément. Mais aussi d’un wallet vers un compte bancaire.

Les wallets disponibles sont: Orange Money, wave, free money, wizall.

Non l’option d’envoi multiples sera bientôt disponible. Pour le moment, pour chaque transfert vous avez un bénéficiaire.

Le transfert vous coûtera 3% par transaction quelque soit le wallet utilisé. Aucun frais supplémentaire.

La plateforme YAWI est une solution développée par COK-SERVICE qui est certifié PCI DSS level 1. Vos transactions sont sécurisées à 100%. La protection des données figure également au cœur de notre action.

La devise utilisée est le FCFA.

Le minimum que vous pouvez envoyer est 500 Fcfa. Le maximum dépend du plafond autorisé par les wallets sélectionnés pour envoyer et recevoir.

Pour annuler une transaction, il faut contacter dès constatation de votre erreur le support de YAWI au +221 77 156 71 90. L’équipe vous guidera afin que vous puissiez annuler la transaction à condition que l’argent transféré ne soit retiré par le bénéficiaire.
